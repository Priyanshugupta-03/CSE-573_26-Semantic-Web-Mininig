## Overview

Common stonecrop is a flowering plant that is native to all of Europe, western Siberia, and North America ( 18 ).

There is insufficient reliable information available about the safety of common stonecrop.

PREGNANCY AND LACTATION: Insufficient reliable information available; avoid using.

#### General

There is limited reliable information available about the adverse effects of common stonecrop.

#### Gastrointestinal

Orally, common stonecrop can cause vomiting and diarrhea when consumed in very large amounts ( 18 ).

Insufficient Reliable Evidence to Rate
Cough.

Although there has been interest in using oral common stonecrop for cough, there is insufficient reliable information about the clinical effects of common stonecrop for this purpose.

Hypertension.

Although there has been interest in using oral common stonecrop for hypertension, there is insufficient reliable information about the clinical effects of common stonecrop for this purpose.

Warts.

Although there has been interest in using topical common stonecrop for warts, there is insufficient reliable information about the clinical effects of common stonecrop for this purpose.

Wound healing.

Although there has been interest in using topical common stonecrop for wound healing, there is insufficient reliable information about the clinical effects of common stonecrop for this purpose.

More evidence is needed to rate common stonecrop for these uses.

#### Adult
Oral:

Research is limited; typical dosing unavailable.

Topical:

Traditionally, a poultice has been made from the fresh leaves.
